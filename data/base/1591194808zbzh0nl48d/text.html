Официальный браузер для TOR.


Содержит клиент TOR в поддиректории /Browser/TorBrowser/Tor


В принципе, клиент может быть запущен самостоятельно


В директории с tor.exe должен лежать конфигурационный файл "torcc"

Для установки как службы выполнить команду (в cmd.exe от имени администратора)

tor.exe --service install -options -f "C:\PathToTOR\torrc"


Для запуска службы

tor.exe --service start


Для проверки, что TOR запустился и прослушивает порт под Windows можно ввести следующую команду

netstat -aon | findstr ":9050"


(если запуск настроен на порт 9050)


Остановка и удаление службы

tor.exe --service stop

tor.exe --service remove
